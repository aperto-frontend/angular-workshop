# Exercise 1: Angular CLI

This branch has been achieved by performing the following steps:

## Step A

Install the Angular CLI, create the project and run it

```
npm install -g @angular/cli@1.6.3
ng new my-angular-workshop
npm install

ng start
```

## Step B

Open the browser at

```
http://localhost:4200/
```
